LeafView2

A minimalist image viewer for macOS based on Leaflet.js and Tauri.

Image by Myriams-Fotos from Pixabay.

👍 Features

  • Pan & Wheel Zoom
  • Browse the images in a folder

📥 Download

You can download the latest version of LeafView2 from the releases page here:
https://github.com/sprout2000/leafview2/releases

📗 Usage

⌨️ Keyboard Shortcuts

Key Function
J or CmdOrCtrl+→ Next Image
K or CmdOrCtrl+← Previous Image
+ Zoom In
- Zoom Out
0 Reset Zoom
← ↑ ↓ → Pan

🖱 Mouse Operations

Mouse Function
Drag Pan
Wheel Zoom in/out
Double click Reset zoom

🛠 Development

% git clone https://github.com/sprout2000/leafview2.git
% cd leafview2
% npm install

# on development
% npx tauri dev

# on production
% npx tauri build

NOTE: You will need to meet Tauri prerequisites.
